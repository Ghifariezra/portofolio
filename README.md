# Portfolio Ghifari Ezra Ramadhan

Hello! Ini adalah website portfolio pribadi saya, dibuat dengan Next.js. Website ini menunjukkan siapa saya, keahlian saya, dan proyek-proyek yang sudah saya kerjakan.

## Fitur Utama
- Profil singkat dengan foto dan bio  
- Daftar proyek lengkap dengan deskripsi, teknologi, dan link demo  
- Blog pribadi untuk berbagi ilmu  
- Responsif dan performa optimal menggunakan Next.js & React Query

## Cara Menjalankan Lokal
1. Clone repository  
2. Jalankan `pnpm install` untuk install dependencies  
3. Jalankan `pnpm dev` untuk development server  
4. Buka `http://localhost:3000` di browser
